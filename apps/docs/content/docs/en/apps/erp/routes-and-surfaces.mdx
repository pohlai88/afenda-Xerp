### User settings (`/settings/**`)

Personal self-service — **ARCH-USER-001** (not tenant admin):

- Profile · Security · Notifications · Preferences
- Admin tenant settings live under `/system-admin/settings/**` only

</Tab>
<Tab value="System Admin">

### System admin (`/system-admin/**`)

Company-scoped control plane — **ARCH-ADMIN-001**:

- Users · Memberships · Roles · Permissions · Audit · Diagnostics
- Settings hub — General, Notifications, Workspace, Integrations, Members, Security, Billing, Appearance

</Tab>

- Workspace dashboard layout — `api/internal/v1/workspace/dashboard-layout/`
- System admin — audit events, user invite, role assignment
- API governance — `apps/erp/src/server/api/`

</Tab>
