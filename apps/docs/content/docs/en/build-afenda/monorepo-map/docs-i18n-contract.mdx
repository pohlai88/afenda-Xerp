## Supported locales

Docs ships **en**, **zh**, **vi** (Vietnamese), **ms** (Malay — Malaysia), **id** (Indonesian), **th** (Thai), and **fil** (Filipino — Philippines) with directory-based content (`content/docs/{locale}/`).

UI shell strings (search, feedback, home cards, etc.) live in JSON — one file per locale under `src/lib/i18n/locales/` (for example `vi.json`). Page body copy stays in MDX under `content/docs/{locale}/`.
