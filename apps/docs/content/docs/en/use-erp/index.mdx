## Overview

Task-based guidance for **Afenda ERP end users** — people who sign in, select a workspace, and work in governed modules. These guides describe routes and flows as they appear in the product, not implementation details in the monorepo.

Start with sign-in if you are new to a tenant. Tenant administrators and integrators should use **Configure Tenant** or **Integrate** instead.
