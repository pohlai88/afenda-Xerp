## Run locally

```bash
pnpm --filter @afenda/erp dev
```

Open port **3000**. Database and auth env vars may be required — see [Dev Setup](/docs/getting-started/dev-setup).

## Boundary rules

**May depend on:** `packages/*` spine (appshell, database, auth, permissions, kernel, etc.)

**Must not depend on:** `@afenda/docs` (leaf documentation app)
