## Type generation

Contract pages use Fumadocs TypeScript docgen ([integration guide](https://fumadocs.dev/docs/integrations/typescript)):

- **MDX (build-time):** `<auto-type-table path="./src/lib/…" name="ExportName" />` — compiled by `remarkAutoTypeTable` into `TypeTable`.
- **RSC (runtime):** `<AutoTypeTable path="…" name="…" />` — same generator via `@/components/auto-type-table`.
- **Cache:** `.next/fumadocs-typescript` (filesystem; safe for serverless).

TSDoc annotations supported on exported types:

| Tag | Effect |
| --- | --- |
| `@internal` | Hide field from generated table |
| `@remarks \`label\`` | Simplified type label in table |
| `@fumadocsType \`Name\`` | Full type display override |
| `@fumadocsHref #anchor` | Link property type to another table anchor |

`DocsGraphNodeData.neighbors` is hidden with `@internal`; `url` uses `@remarks` for display.
