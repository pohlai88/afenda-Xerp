## Purpose

Fumadocs-powered engineer documentation — onboarding, monorepo map, Applications, and contributing workflow. **Zero** approved `@afenda/*` runtime workspace dependencies (dev-only `@afenda/typescript-config`).

## Run locally

```bash
pnpm --filter @afenda/docs dev
```

Open [http://localhost:3001/docs](http://localhost:3001/docs). ERP does not need to be running for docs work.

## Boundary rules

| Allowed | Prohibited |
| --- | --- |
| MDX under `content/docs/` | Import `@afenda/erp`, `@afenda/ui`, spine packages |
| Docs-owned editorial palette | Copy full ARCH/FDR/inventory tables into MDX |
| Fumadocs UI + editorial blocks | Brand accent in shell chrome |
