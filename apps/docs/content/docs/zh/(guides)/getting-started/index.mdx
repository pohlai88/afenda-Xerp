加入 Afenda monorepo：安装工具链、拉取依赖，并启动文档站或 ERP 开发服务器。开发 `@afenda/docs` 时无需运行 ERP。

## 环境要求

- **Node.js** 20 或更高版本
- **pnpm** 10（monorepo 包管理器）
- **Git** — 用于克隆与贡献流程
- **Docker**（可选）— 在 `@afenda/database` 或 ERP 数据路径上工作时用于本地 Postgres

## 快速上手

1. 克隆仓库并在 monorepo 根目录安装依赖：

   ```bash
   git clone https://github.com/pohlai88/afenda-Xerp.git
   cd afenda-Xerp
   pnpm install
   ```

2. 启动文档站（端口 **3001**）：

   ```bash
   pnpm --filter @afenda/docs dev
   ```

3. 打开 [http://localhost:3001/docs](http://localhost:3001/docs)。

如需 ERP 开发，请继续阅读 [安装](/docs/getting-started/installation) 与 [开发环境](/docs/getting-started/dev-setup)。
