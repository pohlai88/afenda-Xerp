# Environment and authentication

Tenant DevOps operators configure authentication and platform connectivity through governed environment variables. **Never commit secret values** to documentation or source control — use `.env.secret` and `pnpm env:sync`.

## Auth-related variables

Review Supabase public keys, Better Auth URL, and OAuth client IDs in `.env.config`. Pair secrets (signing keys, database passwords) from `.env.secret` only on deployment targets.
