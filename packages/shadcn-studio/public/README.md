# @afenda/shadcn-studio public assets

Static files served by Storybook at `/studio-assets/*` and available for promoted blocks that must not depend on vendor CDN URLs at accept time.

**Usage in blocks:** reference `/studio-assets/<path>` after copying assets here during stabilization.

**Default MCP blocks** may still use `cdn.shadcnstudio.com` until promotion replaces them with local copies.
