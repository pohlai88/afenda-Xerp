## Rules frozen

1. Route, query, header, and body canonical enterprise IDs must pass through `parseRoute*` / kernel `parse*` — never unchecked casts.
2. Tenant human references (`EMP-000123`, `SKU-001`, …) must reject when supplied to canonical enterprise ID parsers (§4.1.13).
3. Wire tenant canonical IDs (`ten_*`) resolve once to internal uuid PK via `tenants.enterprise_id` lookup for RLS/session — RLS never parses `ten_*`.
4. Execution and authorization layers keep canonical `TenantId`; database RLS session uses uuid PK only.
5. Family-unknown wire ingress uses `parseWireRegisteredCanonicalId` — not generic regex alone.
