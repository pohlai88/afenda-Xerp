## Rules frozen

1. `REGISTRY_FAMILY_COUNT` and `ID_FAMILY_COUNT` count **enterprise families only** (22).
2. `PRIMITIVE_REFERENCE_COUNT` counts primitive references (7) — not merged into `ID_FAMILIES`.
3. `ID_FAMILY_CATEGORIES` has five enterprise categories — no `global-primitive` category.
4. `ENTERPRISE_ID_TYPE_NAMES` must not include primitive type names (`LocaleCode`, etc.).
5. `FORBIDDEN_PLATFORM_FLOOR_ID_SYMBOLS` remain off the enterprise registry floor.
